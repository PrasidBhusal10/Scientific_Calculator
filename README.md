# 🧮 Scientific Calculator

This is a **fully functional scientific calculator** capable of performing a wide range of mathematical operations, including:

- Basic operations: Addition, Subtraction, Multiplication, Division  
- Algebraic and power operations  
- Trigonometric functions (sin, cos, tan, etc.)  
- Logarithmic and exponential functions  
- Advanced mathematical calculations  

---

## 🚀 Why I Built This Project

The main motivation behind this project was my **curiosity** and my desire to apply my programming skills in **HTML, CSS, and JavaScript**.

I especially enjoy **trigonometry**, so I wanted to build a calculator that focuses more on trigonometric functions. I also plan to use this calculator for solving my own problems.

Additionally, I wanted to make this tool available online so that **others can use it freely and easily without any hassle**.

---

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript  

---

## ⚙️ Development Process

The development of this scientific calculator began with an initial idea and planning phase, where I explored how to build a fully functional calculator using HTML, CSS, and JavaScript.

I started by creating the basic structure with HTML, designing the layout that includes buttons, a display screen, and other essential components. After completing the structure, I moved on to CSS to style the interface by adding colors, adjusting dimensions, setting margins, and implementing hover effects to improve the overall user experience.

Once the design was complete, I used JavaScript to add functionality, writing the logic required for performing calculations and handling user interactions.

Finally, I refined the project by improving responsiveness and making final design adjustments, ensuring that the calculator works smoothly and is user-friendly.

## ▶️ How to Run This Project

To run this project on your local machine, follow these steps:

1. **Clone the repository:**

git clone https://github.com/PrasidBhusal10/Scientific_Calculator.git


2. **Navigate to the project folder:**

cd Scientific_Calculator


3. **Open the project:**
- Locate the file `Calculator.html`
- Open it in any web browser (Chrome, Edge, etc.)

4. **Start using the calculator 🎉**
